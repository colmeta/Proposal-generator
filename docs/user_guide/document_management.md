# Document Management Guide

This guide covers managing generated proposals, versions, and documents in the Proposal Generator.

## Overview

The document management system allows you to:
- View generated proposals
- Download documents in multiple formats
- Manage document versions
- Compare different versions
- Organize and search documents

## Document Viewer

### Accessing Documents

1. Navigate to **Documents** in the main menu
2. View list of all documents
3. Select a document to view details
4. Use filters to find specific documents

### Document Information

Each document displays:
- **Title**: Proposal title
- **Type**: Document type (proposal, revision, etc.)
- **Status**: Generation status
- **Created Date**: When document was created
- **Updated Date**: Last modification date
- **File Size**: Document size
- **Version**: Version number
- **Proposal ID**: Associated proposal

### Viewing Documents

#### Preview Mode
- View document content in browser
- Scroll through sections
- Check formatting
- Review content

#### PDF View
- View PDF version
- Download PDF file
- Print-friendly format
- Preserves formatting

#### DOCX View
- Download Word document
- Editable format
- Compatible with Microsoft Word
- Preserves structure

#### Text View
- Plain text version
- Easy to copy
- Searchable content
- Minimal formatting

## Downloading Documents

### Available Formats

#### PDF
- **Best for**: Final submission, printing, sharing
- **Features**: Preserves formatting, not editable
- **Size**: Usually smaller than DOCX

#### DOCX
- **Best for**: Editing, collaboration, revisions
- **Features**: Editable, compatible with Word
- **Size**: Usually larger than PDF

### Download Process

1. Select document
2. Choose format (PDF or DOCX)
3. Click download button
4. File downloads to your device

### Download Options

- **Single Document**: Download one document
- **Batch Download**: Download multiple documents
- **All Versions**: Download all versions of a document
- **Selected Sections**: Download specific sections (future feature)

## Version Management

### Document Versions

Each time a proposal is regenerated:
- A new version is created
- Previous versions are preserved
- Version history is maintained
- Versions can be compared

### Version History

View version history to see:
- Version numbers
- Creation dates
- Changes made
- Status of each version

### Comparing Versions

#### Side-by-Side Comparison
1. Select two versions
2. Click "Compare Versions"
3. View differences highlighted
4. See what changed

#### Version Details
- View metadata for each version
- Check file sizes
- See creation dates
- Access individual downloads

### Managing Versions

- **View**: Browse all versions
- **Download**: Get specific versions
- **Compare**: See differences
- **Delete**: Remove old versions (with caution)

## Document Organization

### Filtering

Filter documents by:
- **Proposal ID**: Find documents for specific proposal
- **Status**: Filter by generation status
- **Date Range**: Find documents by date
- **Type**: Filter by document type

### Searching

Search documents by:
- **Title**: Search in document titles
- **Content**: Search in document content
- **Keywords**: Search by keywords
- **Metadata**: Search in document metadata

### Sorting

Sort documents by:
- **Date Created**: Newest or oldest first
- **Date Updated**: Recently updated first
- **Title**: Alphabetical order
- **Size**: File size
- **Status**: By generation status

## Best Practices

### Organization
1. **Use Descriptive Titles**: Make documents easy to find
2. **Add Keywords**: Help with searching
3. **Keep Versions**: Don't delete old versions immediately
4. **Organize by Project**: Group related documents

### Version Control
1. **Review Before Regenerating**: Check if changes needed
2. **Compare Versions**: See what improved
3. **Keep Important Versions**: Don't delete milestones
4. **Document Changes**: Note what changed and why

### Storage
1. **Download Important Documents**: Keep local copies
2. **Backup Regularly**: Don't rely solely on system
3. **Organize Locally**: Use folders and naming conventions
4. **Archive Old Documents**: Move completed projects

## Troubleshooting

### Common Issues

#### Document Not Loading
- **Cause**: File corruption or missing
- **Solution**: Regenerate document
- **Prevention**: Check job status before viewing

#### Download Fails
- **Cause**: Network issue or file too large
- **Solution**: Retry download, check connection
- **Alternative**: Use different format

#### Version Missing
- **Cause**: Deleted or not created
- **Solution**: Check version history
- **Prevention**: Don't delete versions prematurely

#### Format Issues
- **Cause**: Conversion problems
- **Solution**: Try different format
- **Report**: Contact support if persistent

### Getting Help
- Check document status
- Review error messages
- Try different format
- Contact support if needed
